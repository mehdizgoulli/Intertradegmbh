BRAND LOGOS — how to add them
================================

Put each brand logo image in THIS folder (assets/brands/) using the exact
file names below. PNG with a transparent background works best. SVG also works.

The website loads each logo automatically. If a file is missing it tries the
Clearbit logo service, and if that fails too it shows the brand name as text.

File names expected (slug):
  ferrero.png
  milka.png
  nestle.png
  mars.png
  lindt.png
  ritter-sport.png
  haribo.png
  oreo.png
  kitkat.png
  toblerone.png
  red-bull.png
  mondelez.png
  storck.png
  loacker.png
  kinder.png
  pringles.png
  toffifee.png
  merci.png
  bahlsen.png
  capri-sun.png

Tips:
- Transparent PNG (or SVG) looks cleanest on the white tiles.
- Recommended size: about 200 px tall is plenty.
- You can also use .svg instead of .png (e.g. ferrero.svg) — both are supported.
